# Özel Anılarımız ❤️

Sevdiğim kişi için hazırladığım özel anılar koleksiyonu web sitesi.

## Nasıl Özelleştirirsiniz?

### 1. Fotoğraf Eklemek
- `index.html` dosyasındaki `.photo-placeholder` div'lerini gerçek `<img>` etiketleriyle değiştirin
- Fotoğraflarınızı projeye ekleyin ve yolları güncelleyin

### 2. Tarihleri Güncellemek
- `script.js` dosyasındaki `startDate` değişkenini ilk buluşma tarihinizle güncelleyin
- `index.html` dosyasındaki timeline tarihlerini doldurun

### 3. İçeriği Kişiselleştirmek
- Tüm metinleri kendi hikayenize göre düzenleyin
- İsimleri, anıları ve mesajları özelleştirin

## GitHub Pages ile Yayınlama

1. Bu repository'yi GitHub'a push edin
2. Repository ayarlarından Pages bölümüne gidin
3. Source olarak `main` branch ve `/ (root)` seçin
4. Siteniz `https://kullaniciadi.github.io/repo-adi` adresinde yayına alınacak

## QR Kod Oluşturma

Siteniz yayına alındıktan sonra:
- https://qr-code-generator.com adresine gidin
- Site URL'nizi girin
- QR kodu indirin ve hediye edin!
