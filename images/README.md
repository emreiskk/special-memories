## Önemli Notlar:

- Fotoğraflar **4:3 oranında** görünecek (yatay: 800x600px veya 1200x900px önerilir)
- JPG, PNG veya JPEG formatında olabilir
- Dosya isimleri küçük harf olmalı (foto1.jpg, foto2.jpg...)
- Türkçe karakter kullanmayın

## Fotoğraf Ekleme:

1. Bu klasöre fotoğraflarınızı atın
2. İsimleri düzenleyin (foto1.jpg, foto2.jpg...)
3. Git ile yükleyin:
   ```bash
   git add images/
   git commit -m "Fotoğraflar eklendi"
   git push
   ```
